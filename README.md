# tauri-calc

An simple calculator built with [Tauri](https://github.com/tauri-apps/tauri).

## Development

Follow instrucions of [Tauri Docs](https://tauri.studio/en/docs/getting-started/intro) to setup environment.

```shell
$ pnpm dev
$ pnpm tauri dev
```

## Build

```shell
$ pnpm build
$ pnpm tauri build
```
